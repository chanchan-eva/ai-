# 数据说明

## `research-stats.json`

来源：`不同LLM对魏晋文本分析文化偏差的数据统计.xlsx`

- 工作表：`Sheet1`
- 原始范围：`A1:L46`
- 观察单位：一篇文本 × 一个模型回答
- 文本数：15
- 回答记录：45
- 模型：GPT 5.5、Claude 4.6、Gemini 3.1
- 偏差字段：过度现代化解释、西方文化框架代入、历史背景遗漏、典故误读、情感倾向偏差、其他偏差

JSON 中同时保留逐条记录与按模型、文本类别、偏差类型汇总后的统计结果，方便复核。

## `cases.json`

四个案例按“每个文本类别中，三个模型偏差出现次数合计最高”选取。模型回答摘要来自对应 Word 原始回答；人工分析结合 Excel 中的偏差次数和具体说明整理。

网站不展示完整模型回答。三个 Word 文档继续作为原始实验记录单独保存。
